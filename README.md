# 🧠 Design Patterns Playground

This project was created to practice **Design Patterns** using **TypeScript** and validating behavior with **Jest**.

---

## 📁 Structure

```
src/
└── patterns/
    ├── factory/
    │   └── ... pattern files
    ├── singleton/
    │   └── ... pattern files
    └── ... other patterns
```

---

## ✅ Implemented Patterns

- [x] Abstract Factory
- [x] Adapter
- [x] Bridge
- [x] Builder
- [x] Composite
- [x] Factory Method
- [x] Singleton

## 🚀 How to Run

### 1. Install dependencies

```bash
npm install
```

### 2. Run tests

```bash
npm test
```
